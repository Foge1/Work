# GruzchikiApp v5.0

Android-приложение для управления грузчиками. Kotlin + Jetpack Compose. Бирюзовая тема.

## Что нового в v5.0

### Для обеих ролей
- ✅ **Тёмная тема** — работающий переключатель в Настройках, сохраняется в DataStore
- ✅ **Настройки** — реальное сохранение уведомлений/звука/вибрации (DataStore)
- ✅ **Обновлённый экран Рейтинга** — аватар с инициалом, звёзды, анимированные карточки статистики
- ✅ **История** — сводная карточка с итогами (кол-во заказов + заработок)
- ✅ **Новый дизайн экрана входа** — карточки ролей с анимацией, иконки, улучшенный UX

### Для диспетчера
- ✅ **Поиск заказов** — строка поиска по адресу и описанию груза с debounce 300мс
- ✅ **Статистика** — кол-во завершённых и активных заказов на экране Рейтинга
- ✅ **Улучшенный диалог создания** — поле «Кол-во часов», поле «Комментарий», расчёт суммы
- ✅ **Карточки заказов** — показывают комментарий и расчётную сумму (₽/час × часов)
- ✅ **Улучшенные пустые состояния** — иконки вместо текста

### Для грузчика
- ✅ **Диалог оценки** — после завершения заказа появляется диалог с 5 звёздами
- ✅ **Статистика** — выполнено заказов, заработано ₽, средний рейтинг
- ✅ **Карточки заказов** — расчётная сумма, отображение уже выставленной оценки
- ✅ **Улучшенные пустые состояния** — иконки, подсказки

## Архитектура
- MVVM + Room + DataStore + Coroutines/Flow
- Jetpack Compose Material 3
- Бирюзовая цветовая схема (light + dark)

## Роли
- **Диспетчер**: создаёт заказы, управляет статусами, видит статистику
- **Грузчик**: принимает заказы, завершает, оценивает
